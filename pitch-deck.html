Technical Android Architecture

App Entry Point
MainActivity.kt + Application Class NavHost (Jetpack Navigation) Deep Link Handling + Intent Filters
UI Layer (Presentation)
Jetpack Compose + Material3 TrueTap Design System (animations, theming, components) Custom Tappy UI Components + Haptic Feedback State Management via Compose State
ViewModel Layer (MVVM)
Kotlin ViewModels + StateFlow Kotlin Coroutines + Flow for reactive streams UI State Management + Event Handling Screen-specific ViewModels (WalletVM, PaymentVM, etc.)
Domain Layer
Clean Architecture UseCases Business Logic + Validation Rules SendPaymentUseCase, SwapTokenUseCase, AuthUseCase Transaction Processing + Crypto Logic
Data Layer (Repository Pattern)
Repositories: Wallet, Contact, Transaction, Token Local: Room Database + DataStore Preferences Remote: Solana RPC + Seed Vault APIs DTO ↔ Domain Model Mappers + Caching
Platform + External SDK Layer
Solana Mobile Stack (Seed Vault SDK) Solana RPC Client + Web3.js Integration Hardware: Bluetooth, NFC, Biometrics, Camera Jupiter Aggregator + Metaplex + Token Extensions
Shared Core / Utilities
Hilt Dependency Injection Navigation Component + Deep Links Logging, Analytics, Crashlytics, Feature Flags Security Utils + Encryption + Key Management
Testing / Quality Assurance
Unit Tests + Integration Tests Compose UI Tests + Screenshot Testing Test Dispatchers + Mock Repositories Automated E2E Testing + Performance Monitoring

Built with: Kotlin 2.0 • Jetpack Compose • Hilt DI • Solana Mobile Stack • Room DB • Coroutines • Material3 • Seed Vault SDK
